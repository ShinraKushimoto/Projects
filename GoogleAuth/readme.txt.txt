1. extract the file googleAuth to localhost
2. create a database name googleauth
3. open user.sql and copy the code or script to create a table
4. change the username and password in config.php file base on you
    username and password of your database
5. execute the program and register your name, email add, username and password
6. install the Google Authenticator to your mobile phone
7. active Google Authenticator mobile App and click Add (+) to scan a QR code
8. enter the key generated by the Google Authenticator to login to the web App
9. logout and relogin
10. open again the Google Athenticator to check the new key generated by the App

Note: the key will expire every 30 Seconds
